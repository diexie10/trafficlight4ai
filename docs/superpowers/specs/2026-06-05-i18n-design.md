# 国际化（i18n）设计文档

## 概述

为 trafficlight4ai 添加国际化支持，支持英语（默认）、中文、日语三种语言。使用 Qt 标准 i18n 机制（QTranslator + .ts/.qm），在设置对话框中切换语言后立即生效。

## 方案

Qt 标准 i18n：所有 UI 字符串用 `tr()` 包裹，`.ts` 翻译文件编译为 `.qm`，运行时通过 `QTranslator` 加载，切换时调用 `retranslateUi()` 刷新。

## 翻译文件

`translations/` 目录下 3 个 `.ts` 文件：

- `trafficlight4ai_en.ts` — 英语（源语言）
- `trafficlight4ai_zh.ts` — 中文
- `trafficlight4ai_ja.ts` — 日语

CMake 中用 `qt_add_translations` 编译为 `.qm` 并嵌入 qrc。

## 源码改造

所有硬编码 UI 字符串改为 `tr("English source")`，当前中文文本替换为英文源 + 翻译文件。

## ConfigManager

新增 `"language": "en"`，可选 `"zh"`、`"ja"`。`language()` / `setLanguage()` 接口。

## 运行时切换

1. 用户在 SettingsDialog 中切换语言 QComboBox
2. `onLanguageChanged` slot → `ConfigManager::setLanguage()`
3. 卸载旧 QTranslator，加载新 `.qm`，安装到 QApplication
4. 调用 `retranslateUi()` 刷新所有控件文本

## retranslateUi()

| 类 | 刷新内容 |
|---|---|
| SettingsDialog | 窗口标题、label、按钮、combo 选项 |
| TrayIcon | 菜单项、tooltip |
| FloatingWindow | 右键菜单（每次弹出动态创建，自动用新语言） |
